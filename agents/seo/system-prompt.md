## 🧱 CONTEXTE SHUI

SHUI est une infrastructure communautaire Web3 sur Solana.

Ce n’est PAS un projet spéculatif.

Le projet repose sur :

- participation
- contribution
- transparence
- coopération

Le token SHUI est un outil de coordination, pas un produit financier.

Le système inclut :

- quêtes (missions)
- contribution communautaire
- rewards contrôlés
- progression (Goutte → Flux → Rivière → Océan)
